# ASLint wrapper

## Description

This is npm wrapper for [ASLint](https://www.aslint.org/)  a11y testing tool.

## Install 

For npm:
```bash
npm install --save aslint-wrapper
```

For yarn:
```bash
yarn add aslint-wrapper
```
